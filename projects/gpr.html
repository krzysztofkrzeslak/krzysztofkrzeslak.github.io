Vacay

This project was about redesign/improvement of existing Ground-penetrating radar hardware and software, which at the starting point was inconvenient to use. The reason for that was that for operation it required external PC and battery and software was not designed to handle any hardware problems.

Scope of work

The first part of work was getting rid of the need to use external equipment to operate, so existing electronics were moved to the new case where complete battery along with PC(in an industrial panel format, running Windows OS) was embedded Starting point

Orginal software utilized Swing Library for UI and JNI for communication with the hardware layer. It needed adaptations to new hardware along with a lot of bugfixes and UX improvements.
